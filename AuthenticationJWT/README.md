## The APIs in the project
* Get all books (**`GET`**)
* Get a book by Id (**`GET`**)
* Create a book (**`POST`**)

* Update a book with PATCH method (**`PATCH`**)
* Delete a book by Id (**`DELETE`**)
* Login (**`POST`**)
* Register (**`POST`**)
